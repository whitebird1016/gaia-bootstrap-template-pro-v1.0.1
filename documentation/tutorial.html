Tutorial

Short Description and Usage

Gaia is a beautiful resource built over the Bootstrap framework. If you are looking for a fast and stylish way to get started on your site, this is the perfect place to start. We are proud of this new Bootstrap skin and the possibilities for customisation.

Using it is very simple and it will enable you to refresh you classic looking application.

Getting Started

Gaia Bootstrap Template is built on top of Bootstrap 3, so you can safely use it on your existing or new Bootstrap project. No line of code from Bootstrap 3 was changed, so you don't have to worry about undesired effects in your work.

We provide all the necessary CSS resources. To get going, just include assets/css/gaia.css in your HTML template. Your project will get the new look.

The easiest way to start is to use our start-up template where all the files are already included and ready to use.
